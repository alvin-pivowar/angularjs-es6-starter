Overview

This Angular starter project uses:

  • AngularJS 1.x
  • angular-route
  • angular-ui-bootstrap
  • Bootstrap CSS 3.x
  • Font Awesome
  • LESS
The Javascript is written using ECMAScript 6; the babel transpiler is used. All of the parts are packaged by webpack. This allows the use of import, export, and require.

Refer to the other tabs to see how ES6 classes can be used to define AngularJS recipes.
