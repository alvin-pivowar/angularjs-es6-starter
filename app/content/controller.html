Controllers

class MyController {
    static get name() { return "my controller name"; }

    constructor($inject1, inject2) {
    }

    static factory($inject1, inject2) { return new MyController($inject1, inject2); }
}

MyController.$inject = ["$inject1", "inject2", MyController.factory];

The controller class has static members that define the recipe name of the controller, and a factory method for building it. The $inject array is used by AngularJS when creating an instance of the controller. To register the controller (from the class definition:

angular.module("myModule").controller(MyController.name, MyController.$inject);
